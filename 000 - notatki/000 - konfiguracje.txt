VS Code

podstawowe komendy

ctrl + shift + P    --  paleta poleceń
ctrl + `            --  terminal
ctrl + \            --  dzielenie ekranu edytora na dwa (domyślnie: dwa pionowe okienka)
ctrl + alt + \      --  dzielenie ekranu edytora na dwa poziome okienka
podzielić ekran edytora można również przez wybranie: wyświetl -> układ edytora
a także (trzymając wciśnięty klawisz alt) klikając na ikonkę w prawym górnym rogu
bądź przeciągając/przenosząc "kartę" z otwartym dokumentem
ctrl + /            --  komentowanie "wierszowe"
ctrl + shift + /    --  komentowanie "blokowe"
